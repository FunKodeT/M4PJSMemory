|###########################################################|
|###########WELCOME TO THIS JAVASCRIPT MEMORY GAME!!########|
|###########################################################|
| You will be flipping cards in this game in order to match	|
| 	the image on one flipped card with a second flipped		|
	card.													|
| The rules are simple, the game is fun, and your memory	|
|	will become stronger through this simply fun memory		|
|	game!!!													|
|-----------------------------------------------------------|
|-----------------------HOW TO PLAY:------------------------|
|-----------------------------------------------------------|
| Step 1) Using the cursor in the browser, click on your 	|
|	chosen card.											|
| Step 2) Move to a different chosen card and click on it	|
| Step 3) If the cards are a match, they will stay flipped.	|
|	If they are still flipped, choose two more cards to 	|
|	match. Try to have all the cards flipped to win!		|
| Step 4) If the cards were not a match, keep choosing new	|
|	ones while trying to remember previous cards and their	|
|	images. Keep trying to find matches, and if you find a	|
|	card that matches an unflipped card you know the		|
|	location of, give them a click to get them matched!		|
|###########################################################|
|###########################ENJOY!##########################|
